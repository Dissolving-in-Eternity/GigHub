# GigHub
Mini social network for live music lovers.
***
## Tech Stack
### Backend
* ASP.NET MVC 5
* ASP.NET Web API 2
* Entity Framework Code First
* Ninject
* AutoMapper
### Frontend
* HTML, CSS (Less), JS
* Bootstrap 3
* jQuery (+ AJAX)
* PagedList.Mvc
* Bootbox.js
* Moment.js
* Underscore.js
* Animate.css

### Testing
* MSTest
* NUnit
* Moq
* Fluent Assertions
***
